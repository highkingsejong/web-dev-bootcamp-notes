css selector

*은 전체 선택자로 모든 요소를 선택합니다. 반점을 써서 두개를 한번에 적용가능 #아이디로 선택할때는 #을 붙여서 선택합니다. id는 최소한으로 써야한다 class등 다른 방법도 많기 때문에 .클래스는 앞에 .을 붙여서 선택합니다. 띄어쓰기는 자식 선택자로 자식 요소를 선택합니다. li a 는 li 안에 있는 앵커만 선택합니다. 이 아래 빈도 낮음 인접선택자+는 바로 다음에 오는 요소를 선택합니다. h1 + p는 h1 다음에 오는 p를 선택합니다. 직계자손 선택자 >는 자식 요소를 선택합니다. div > p는 div의 자식인 p를 선택합니다. 한단계 아래만 인정합니다. 속성 선택자는 []를 사용합니다. ex a[href=""], input[type="text"]는 type이 text인 요소를 선택합니다. 가상 클래스 는 앞에 :를 붙입니다. id선택자에는 충돌이 되어 변하지 않음 :hover는 마우스가 올라갔을때 선택합니다. :focus는 포커스가 갔을때 선택합니다. :active는 클릭했을때 선택합니다. :checked는 체크박스가 체크되었을때 선택합니다. :nth-of-type(2n)은 2의 배수인 요소를 선택합니다. ::가상요소는 ::를 붙입니다. ::first-letter는 첫번째 글자를 선택합니다. ::first-line은 첫번째 줄을 선택합니다. ::selection은 드래그한 부분을 선택합니다. casecade 같은 내용이 있으면 아래 것이 덮어씌워 적용됨 특이도는 id > class > tag > 요소 순으로 적용됨 반대로 선점한걸 바꿀 수 없음 하지만 인라인 스타일이 있으면 그게 우선 적용됨 !importaant는 제일 와따임 제일 우선순위가 높음 하지만 인라인스타일과 임포턴트는 사용하지 않는게 좋음 특이도를 무시하기 때문 css 상속 부모에 적용된 css가 자식에게도 적용됨 하지만 모든 css가 상속되는건 아님 버튼이랑 인풋은 상속이 안됨 상속받고싶으면 안에 :inherit을 붙어야댐
